Proyecto final; Conceptos Tics

1. Hardware

Es la parte física de un sistema informático: computadoras, servidores, teléfonos móviles, impresoras, entre otros. Sin el hardware no habría un soporte donde ejecutar programas ni procesar información.

2. Software

Son los programas y aplicaciones que permiten al hardware realizar tareas específicas, desde sistemas operativos hasta apps móviles. Sin software, los dispositivos serían solo piezas inertes.

3. Sistema Operativo

Es el software principal de un dispositivo que gestiona los recursos y permite la interacción entre el usuario y el hardware. Ejemplos: Windows, Linux, macOS, Android.

4. CPU (Unidad Central de Procesamiento)

Es considerado el “cerebro” del computador. Se encarga de interpretar y ejecutar instrucciones, haciendo posible el funcionamiento de todos los procesos.

5. Memoria RAM

Es la memoria de acceso rápido donde se almacenan temporalmente los datos que se están utilizando. Cuanta más RAM tenga un dispositivo, más rápido y eficiente puede trabajar.

6. Disco Duro / SSD

Es el dispositivo de almacenamiento donde se guardan permanentemente programas, documentos y archivos. Los SSD son más rápidos y modernos que los discos duros mecánicos.

7. Redes Informáticas

Conjunto de dispositivos interconectados que comparten información y recursos. Pueden ser pequeñas (LAN) o extensas (WAN, Internet).

8. Internet

Es la red global que conecta millones de computadoras y dispositivos en todo el mundo. Permite compartir información y servicios como correo electrónico, redes sociales y la web.

9. Dirección IP

Es un número único que identifica un dispositivo en una red. Puede ser IPv4 (ej. 192.168.0.1) o IPv6 (más moderna y extensa).

10. DNS (Sistema de Nombres de Dominio)

Traduce las direcciones IP en nombres fáciles de recordar como www.google.com . Sin el DNS, tendríamos que memorizar números para navegar en internet.

11. Router

Es el dispositivo que conecta varias redes entre sí, como la red local de una casa con Internet. Administra el tráfico de datos para que llegue a su destino correcto.

12. Switch

Permite la interconexión de varios dispositivos en una red local, gestionando la comunicación interna. Es esencial en redes empresariales.

13. Firewall

Es un sistema de seguridad que controla el tráfico de red entrante y saliente. Protege a los equipos contra accesos no autorizados y ataques externos.

14. Antivirus

Programa diseñado para detectar, prevenir y eliminar virus informáticos. Actualmente también protegen contra malware, spyware y phishing.

15. Nube (Cloud Computing)

Es el almacenamiento y procesamiento de datos a través de servidores en Internet, sin necesidad de mantener toda la infraestructura local. Ejemplo: Google Drive, OneDrive.

16. Correo Electrónico

Es un servicio que permite enviar y recibir mensajes digitales a través de Internet. Es uno de los medios de comunicación más utilizados en entornos personales y laborales.

17. Redes Sociales

Plataformas digitales que facilitan la interacción, el intercambio de información y la creación de comunidades. Ejemplos: Facebook, Instagram, LinkedIn.

18. Big Data

Se refiere al manejo de grandes volúmenes de datos que no pueden procesarse con herramientas tradicionales. Es clave en el análisis y toma de decisiones.

19. Inteligencia Artificial (IA)

Conjunto de tecnologías que permiten a las máquinas aprender, razonar y resolver problemas de forma similar al ser humano. Se aplica en asistentes virtuales, diagnósticos médicos y más.

20. Ciberseguridad

Disciplina que busca proteger sistemas, redes y datos frente a ataques, accesos no autorizados y vulnerabilidades. Es esencial en un mundo cada vez más digitalizado.

21. HTML (HyperText Markup Language)

Es el lenguaje estándar para crear páginas web. Define la estructura básica de un sitio usando etiquetas que organizan texto, imágenes y enlaces.

22. CSS (Cascading Style Sheets)

Es el lenguaje que da estilo a las páginas web: colores, tamaños, tipografía y diseño visual. Se complementa con HTML para mejorar la presentación.

23. JavaScript

Es un lenguaje de programación que permite dar interactividad a las páginas web, como animaciones, validaciones de formularios y menús dinámicos.

24. Bases de Datos

Son sistemas organizados para almacenar, gestionar y recuperar información. Pueden ser relacionales (SQL) o no relacionales (NoSQL).

25. SQL (Structured Query Language)

Es el lenguaje de programación que se usa para gestionar y consultar datos en bases de datos relacionales, como MySQL, PostgreSQL o SQL Server.

26. Backup (Copia de Seguridad)

Es una copia de los datos importantes para evitar pérdidas en caso de fallas técnicas, ataques o borrados accidentales.

27. Malware

Término general para referirse a software malicioso como virus, gusanos, troyanos y ransomware, diseñado para dañar o robar información.

28. Phishing

Es una técnica de estafa digital que busca engañar a las personas para que revelen información confidencial, simulando ser entidades confiables.

29. Criptografía

Es el arte de proteger la información mediante técnicas de codificación. Garantiza confidencialidad y seguridad en las comunicaciones.

30. VPN (Red Privada Virtual)

Es una tecnología que crea un canal seguro y encriptado para navegar en internet de manera privada, ocultando la dirección IP real del usuario.

31. Ethernet

Es una tecnología que permite la conexión de dispositivos en redes locales mediante cables. Es rápida y muy usada en entornos empresariales.

32. Wi-Fi

Es una tecnología inalámbrica que permite conectar dispositivos a internet o a una red local sin necesidad de cables.

33. Bluetooth

Es un protocolo de comunicación inalámbrica de corto alcance, usado para conectar periféricos como audífonos, teclados y dispositivos móviles.

34. IoT (Internet de las Cosas)

Se refiere a la interconexión de objetos cotidianos a través de internet, como relojes inteligentes, refrigeradores o autos conectados.

35. Blockchain

Es una tecnología de registro distribuido que garantiza transparencia y seguridad en transacciones digitales, usada principalmente en criptomonedas.

36. Criptomonedas

Son monedas digitales basadas en criptografía y blockchain, como Bitcoin o Ethereum, que permiten transacciones seguras sin intermediarios.

37. Realidad Virtual (VR)

Es una simulación digital inmersiva que coloca al usuario en un entorno generado por computadora, interactuando con él mediante gafas o sensores.

38. Realidad Aumentada (AR)

Superpone elementos digitales al mundo real mediante dispositivos como celulares o gafas inteligentes, ampliando la experiencia del usuario.

39. 5G

Es la quinta generación de redes móviles, que ofrece mayor velocidad, menor latencia y permite la conexión masiva de dispositivos IoT.

40. Metaverso

Es un espacio virtual compartido, en desarrollo, donde los usuarios interactúan en entornos 3D con avatares digitales, combinando VR, AR e internet.

41. Lenguaje de Programación

Es un conjunto de instrucciones y reglas que permiten a los programadores comunicarse con una computadora. Ejemplos: Python, Java, C++.

42. Python

Es un lenguaje de programación popular por su simplicidad y versatilidad. Se usa en desarrollo web, ciencia de datos, inteligencia artificial y automatización.

43. Java

Lenguaje de programación orientado a objetos, muy usado en aplicaciones empresariales, móviles (Android) y sistemas distribuidos.

44. C++

Lenguaje de programación de alto rendimiento, ideal para videojuegos, sistemas operativos y software que requiere gran control del hardware.

45. Algoritmo

Es un conjunto de pasos ordenados y finitos para resolver un problema o realizar una tarea específica en informática.

46. Compilador

Es un programa que traduce el código fuente de un lenguaje de programación a lenguaje máquina para que la computadora lo ejecute.

47. Intérprete

Programa que ejecuta directamente el código fuente, línea por línea, sin necesidad de compilarlo previamente. Ejemplo: Python.

48. Open Source

Software cuyo código fuente es abierto y puede ser estudiado, modificado y distribuido libremente por cualquier persona.

49. Propietario

Software que pertenece a una empresa o persona y cuyo código no está disponible para modificación. Ejemplo: Microsoft Office.

50. API (Interfaz de Programación de Aplicaciones)

Es un conjunto de funciones y protocolos que permiten que diferentes programas se comuniquen entre sí.

51. SDK (Kit de Desarrollo de Software)

Conjunto de herramientas que facilitan la creación de aplicaciones para un sistema o plataforma específica.

52. Framework

Estructura base de software que ofrece librerías y herramientas para desarrollar aplicaciones más rápido y de manera organizada.

53. Librería

Colección de funciones y recursos reutilizables que los programadores pueden integrar en sus proyectos para ahorrar tiempo.

54. Interfaz Gráfica de Usuario (GUI)

Es la parte visual de un software que permite a los usuarios interactuar mediante ventanas, botones e iconos.

55. Línea de Comandos (CLI)

Es una interfaz basada en texto donde los usuarios escriben comandos para interactuar con el sistema operativo.

56. Virtualización

Tecnología que permite ejecutar varios sistemas operativos o aplicaciones en una misma máquina mediante entornos virtuales.

57. Máquina Virtual

Es un software que simula un sistema operativo independiente dentro de otro, útil para pruebas y seguridad.

58. Servidor

Es una computadora o software que proporciona servicios, recursos o datos a otros dispositivos llamados clientes.

59. Cliente

Es el dispositivo o programa que solicita y consume los servicios proporcionados por un servidor en una red.

60. Arquitectura Cliente-Servidor

Modelo de comunicación en el que un servidor ofrece recursos y un cliente los solicita. Es la base de internet y muchas aplicaciones.

61. Hosting

Es el servicio que permite almacenar un sitio web en servidores conectados a internet, para que sea accesible desde cualquier lugar del mundo.

62. Dominio

Es el nombre único que identifica un sitio web en internet, como www.ejemplo.com, facilitando el acceso sin usar direcciones IP.

63. HTTP

Es el protocolo que permite la transferencia de información en la web. Funciona como base de comunicación entre navegadores y servidores.

64. HTTPS

Es la versión segura de HTTP, que cifra los datos mediante SSL/TLS para proteger la información en las comunicaciones web.

65. SSL/TLS

Son protocolos de seguridad que encriptan la conexión entre el usuario y el servidor, garantizando confidencialidad e integridad de datos.

66. URL

Es la dirección completa que indica la ubicación de un recurso en internet, incluyendo dominio, protocolo y ruta de acceso.

67. Navegador Web

Programa que permite acceder y visualizar páginas web, como Google Chrome, Firefox, Edge o Safari.

68. Motor de Búsqueda

Sistema que indexa páginas web y permite a los usuarios encontrar información fácilmente. Ejemplos: Google, Bing, DuckDuckGo.

69. SEO (Search Engine Optimization)

Técnicas que optimizan un sitio web para mejorar su visibilidad en los resultados de los motores de búsqueda.

70. SEM (Search Engine Marketing)

Estrategias de marketing digital que incluyen publicidad pagada en motores de búsqueda para aumentar la visibilidad de un sitio web.

71. CMS (Content Management System)

Plataforma que permite crear y gestionar sitios web fácilmente sin necesidad de programar. Ejemplos: WordPress, Joomla, Drupal.

72. Wordpress

Es el CMS más popular del mundo, usado para crear blogs, tiendas en línea y páginas web de todo tipo.

73. E-commerce

Se refiere al comercio electrónico, es decir, la compra y venta de bienes o servicios a través de internet.

74. Pasarela de Pago

Servicio que procesa transacciones financieras en línea de forma segura, como PayPal, Stripe o MercadoPago.

75. UX (Experiencia de Usuario)

Se refiere a la percepción que tiene una persona al interactuar con un producto digital. Busca que la navegación sea simple y agradable.

76. UI (Interfaz de Usuario)

Es el diseño visual de un sistema digital: botones, menús, colores y tipografía que facilitan la interacción.

77. Diseño Responsivo

Técnica de diseño web que adapta automáticamente la visualización de un sitio a diferentes dispositivos como celulares, tablets y PC.

78. App

Es una aplicación de software diseñada para dispositivos móviles o de escritorio que cumple funciones específicas.

79. Aplicación Web

Programa que se ejecuta en un navegador sin necesidad de instalarse en el dispositivo, como Gmail o Google Docs.

80. Aplicación Móvil

Es un software diseñado específicamente para ejecutarse en smartphones o tablets, disponible en tiendas como Google Play o App Store.

81. Inteligencia de Negocios (BI)

Conjunto de herramientas y procesos que transforman datos en información útil para la toma de decisiones empresariales.

82. Análisis de Datos

Proceso de examinar datos para descubrir patrones, tendencias y obtener conclusiones relevantes.

83. Ciencia de Datos

Disciplina que combina estadística, programación y conocimiento de negocio para extraer valor de grandes volúmenes de datos.

84. Machine Learning

Rama de la inteligencia artificial que permite a los sistemas aprender de los datos y mejorar su rendimiento sin ser programados explícitamente.

85. Deep Learning

Subcampo del machine learning que utiliza redes neuronales profundas para resolver problemas complejos como visión por computadora o procesamiento de lenguaje.

86. Chatbot

Programa que simula conversaciones humanas, brindando respuestas automáticas a los usuarios mediante texto o voz.

87. Asistente Virtual

Software basado en inteligencia artificial que ayuda a los usuarios con tareas diarias, como Siri, Alexa o Google Assistant.

88. Data Center

Instalación física que alberga servidores y equipos de red para almacenar, gestionar y distribuir datos a gran escala.

89. Cloud Storage

Servicio de almacenamiento de archivos en servidores remotos accesibles por internet, como Google Drive o Dropbox.

90. Edge Computing

Modelo de procesamiento de datos que acerca la computación a los dispositivos de origen para reducir latencia y mejorar la eficiencia.

91. DevOps

Metodología que integra desarrollo de software y operaciones de TI para acelerar la entrega de aplicaciones y servicios.

92. Scrum

Marco ágil de trabajo que organiza proyectos en ciclos cortos llamados sprints, mejorando la productividad y la colaboración.

93. Kanban

Método visual de gestión de proyectos que organiza tareas en tableros con columnas como “Pendiente”, “En proceso” y “Finalizado”.

94. Git

Sistema de control de versiones que permite a los desarrolladores colaborar y mantener un historial de cambios en el código.

95. GitHub

Plataforma en la nube basada en Git que facilita la colaboración en proyectos de software y el almacenamiento de repositorios.

96. Ciberataque

Acción maliciosa que busca dañar, robar o interrumpir sistemas informáticos, redes o datos.

97. Ingeniería Social

Técnicas de manipulación psicológica usadas por atacantes para obtener información confidencial de los usuarios.

98. Ransomware

Tipo de malware que bloquea el acceso a los archivos de un sistema y exige un pago para liberarlos.

99. Backup en la Nube

Copia de seguridad de datos almacenada en servidores remotos accesibles por internet, ofreciendo mayor seguridad y accesibilidad.

100. Transformación Digital

Proceso mediante el cual las organizaciones adoptan tecnologías digitales para mejorar sus procesos, servicios y competitividad.

101. Computación en la Nube Híbrida

Es un modelo que combina la nube pública y la nube privada, permitiendo a las empresas usar lo mejor de ambos entornos.

102. Nube Privada

Infraestructura en la nube que pertenece a una sola organización, ofreciendo mayor control y seguridad sobre los datos.

103. Nube Pública

Servicios en la nube ofrecidos por terceros como Amazon Web Services o Microsoft Azure, accesibles para múltiples usuarios.

104. SaaS (Software as a Service)

Modelo de servicio en la nube donde el software se usa en línea sin necesidad de instalación local. Ejemplo: Gmail.

105. PaaS (Platform as a Service)

Modelo en la nube que ofrece plataformas listas para desarrollar y desplegar aplicaciones sin preocuparse por la infraestructura.

106. IaaS (Infrastructure as a Service)

Servicio en la nube que proporciona infraestructura virtualizada, como servidores y almacenamiento bajo demanda.

107. Colaboración en Línea

Uso de herramientas digitales para trabajar en equipo desde diferentes ubicaciones. Ejemplo: Google Workspace.

108. Teletrabajo

Modalidad laboral que aprovecha las TICs para desempeñar actividades desde cualquier lugar fuera de la oficina tradicional.

109. Videoconferencia

Comunicación en tiempo real mediante video y audio, usada en educación, trabajo remoto y reuniones virtuales.

110. Streaming

Tecnología que permite la transmisión de audio o video en tiempo real sin necesidad de descargar el archivo completo.

111. Podcast

Archivo de audio digital que se distribuye en línea y puede ser escuchado bajo demanda desde diferentes dispositivos.

112. Web 1.0

Primera generación de la web, caracterizada por páginas estáticas con contenido únicamente informativo.

113. Web 2.0

Segunda generación de la web, donde los usuarios no solo consumen, sino también crean y comparten contenido.

114. Web 3.0

Evolución de la web hacia la descentralización, el uso de blockchain y mayor personalización basada en inteligencia artificial.

115. Foros en Línea

Espacios digitales donde los usuarios pueden intercambiar ideas,preguntas y respuestas en torno a temas específicos.

116. Wiki

Sitio web colaborativo donde los usuarios pueden crear, editar y compartir contenido de manera conjunta. Ejemplo: Wikipedia.

117. Blog

Sitio web donde se publican artículos o entradas de manera periódica sobre diferentes temas.

118. Vlog

Versión de un blog en formato de video, muy común en plataformas como YouTube.

119. Influencer Digital

Persona que crea contenido en redes sociales y tiene capacidad de influir en la opinión o decisiones de sus seguidores.

120. Marketing Digital

Uso de estrategias y herramientas tecnológicas para promocionar productos o servicios en internet.

121. Analítica Web

Medición y análisis del tráfico de un sitio web para entender el comportamiento de los usuarios.

122. Cookies

Pequeños archivos que los sitios web almacenan en el navegador del usuario para recordar preferencias y actividad.

123. Chat en Línea

Servicio que permite la comunicación instantánea entre dos o más personas mediante texto en internet.

124. Foro Técnico

Espacio en línea donde especialistas y principiantes resuelven dudas sobre temas tecnológicos.

125. Netiqueta

Conjunto de normas de comportamiento y cortesía en la comunicación digital.

126. Backup Incremental

Copia de seguridad que guarda solo los cambios realizados desde el último respaldo, optimizando espacio y tiempo.

127. Backup Completo

Copia de seguridad que guarda toda la información sin importar si hubo cambios o no.

128. Almacenamiento Externo

Dispositivo físico como discos duros externos o memorias USB que permiten guardar y transportar información.

129. Nube Educativa

Servicios en la nube adaptados al sector académico para gestionar clases, tareas y recursos educativos.

130. Aula Virtual

Espacio digital donde profesores y estudiantes interactúan mediante plataformas como Moodle o Google Classroom.

131. Gamificación

Estrategia que aplica dinámicas de juego en entornos educativos o laborales para motivar a los usuarios.

132. MOOC

Cursos masivos en línea de acceso abierto, diseñados para que cualquier persona pueda aprender a distancia.

133. Inteligencia Colectiva

Capacidad de colaboración y creación conjunta de conocimiento entre comunidades conectadas por internet.

134. Identidad Digital

Conjunto de datos e información que representan a una persona u organización en el entorno digital.

135. Huella Digital

Rastro de información que una persona deja en internet mediante su actividad en redes, búsquedas o publicaciones.

136. Contraseña Segura

Clave que combina letras, números y símbolos para proteger cuentas digitales contra accesos no autorizados.

137. Autenticación de Dos Factores

Método de seguridad que combina la contraseña con un segundo paso de verificación, como un código enviado al móvil.

138. Spam

Mensajes electrónicos no deseados, generalmente de carácter publicitario, enviados masivamente.

139. Bot

Programa que realiza tareas automatizadas en internet, como responder mensajes o indexar páginas web.

140. Deepfake

Tecnología que usa inteligencia artificial para crear videos o imágenes falsas de personas con gran realismo.

141. Ética Digital

Conjunto de principios que guían el comportamiento responsable y seguro en entornos tecnológicos.

142. Derechos de Autor Digitales

Leyes que protegen la propiedad intelectual de contenidos publicados en entornos digitales.

143. Creative Commons

Licencias que permiten compartir obras con diferentes grados de permiso, respetando los derechos de autor.

144. Accesibilidad Web

Prácticas de diseño que aseguran que los sitios web puedan ser usados por personas con discapacidades.

145. Inclusión Digital

Proceso de garantizar que todas las personas tengan acceso y competencias para usar las TICs.

146. Brecha Digital

Diferencia entre personas que tienen acceso a las TICs y aquellas que no, ya sea por factores económicos o geográficos.

147. Alfabetización Digital

Capacidad de una persona para comprender y usar herramientas tecnológicas de manera crítica y responsable.

148. Realidad Mixta

Combinación de realidad virtual y aumentada que permite interactuar con objetos físicos y digitales al mismo tiempo.

149. Gemelos Digitales

Réplicas virtuales de objetos o procesos reales que se usan para pruebas, simulaciones y monitoreo.

150. Computación Verde

Uso responsable y eficiente de la tecnología para reducir el impacto ambiental, optimizando recursos energéticos.

Nivel Principiante

Segun una leve investigacion echa, un especialista en TICS debe conocer y manejar almenos estos primeros 150 conceptos. Si lo hace se puede categorizar en principiante. Ahora continuaremos con conceptos un poco mas avanzados

151. Protocolo TCP/IP

Conjunto de reglas que permiten la comunicación entre dispositivos en internet, gestionando cómo se envían y reciben los datos.

152. Subred

División lógica de una red más grande, que mejora la eficiencia y seguridad del tráfico de datos.

153. Máscara de Subred

Es un valor numérico que define qué parte de una dirección IP pertenece a la red y cuál a los hosts.

154. VLAN

Red de área local virtual que permite segmentar redes físicas en grupos lógicos para mejorar el rendimiento y la seguridad.

155. NAT (Network Address Translation)

Técnica que traduce direcciones IP privadas en públicas, permitiendo a múltiples dispositivos compartir una sola IP pública.

156. DHCP

Protocolo que asigna direcciones IP dinámicas automáticamente a los dispositivos en una red.

157. FTP

Protocolo que permite la transferencia de archivos entre computadoras a través de una red.

158. SFTP

Versión segura de FTP que cifra las transferencias de archivos para proteger la información.

159. SSH

Protocolo que permite acceder y administrar dispositivos de manera segura mediante una conexión encriptada.

160. Telnet

Protocolo antiguo que permite controlar dispositivos de red de forma remota, aunque sin seguridad, hoy reemplazado por SSH.

161. SNMP

Protocolo usado para monitorear y gestionar dispositivos de red como routers, switches y servidores.

162. MPLS

Tecnología que mejora la velocidad y eficiencia del tráfico en redes al enrutar datos mediante etiquetas en lugar de direcciones IP.

163. QoS (Quality of Service)

Conjunto de técnicas que priorizan ciertos tipos de tráfico en la red para garantizar calidad en aplicaciones críticas como VoIP o videoconferencia.

164. Latencia

Tiempo que tarda un paquete de datos en viajar desde su origen hasta su destino en una red.

165. Ancho de Banda

Capacidad máxima de transmisión de datos en una red durante un período de tiempo determinado.

166. Packet Loss

Se refiere a la pérdida de paquetes de datos durante la transmisión, lo que afecta la calidad de la comunicación.

167. DNS Dinámico (DDNS)

Servicio que actualiza automáticamente la dirección IP asociada a un nombre de dominio cuando esta cambia.

168. Proxy

Servidor que actúa como intermediario entre un cliente y un servidor, usado para mejorar seguridad y rendimiento.

169. Balanceador de Carga

Dispositivo o software que distribuye el tráfico de red entre varios servidores para optimizar el rendimiento.

170. CDN (Content Delivery Network)

Red de servidores distribuidos que almacenan copias de contenido web para reducir la latencia y mejorar la velocidad de acceso.

171. IDS (Intrusion Detection System)

Sistema que monitorea la red en busca de actividades sospechosas o ataques.

172. IPS (Intrusion Prevention System)

Similar al IDS, pero además de detectar, bloquea ataques automáticamente.

173. Honeypot

Sistema trampa diseñado para atraer a atacantes y estudiar sus métodos sin comprometer sistemas reales.

174. DMZ (Zona Desmilitarizada)

Segmento de red separado que aloja servicios públicos como servidores web, aislándolos de la red interna.

175. Zero Trust

Modelo de seguridad que no confía en ningún dispositivo o usuario, incluso si está dentro de la red corporativa.

176. DLP (Data Loss Prevention)

Tecnología que previene la fuga o filtración de información sensible de una organización.

177. SIEM

Plataforma que recopila y analiza eventos de seguridad en tiempo real para detectar amenazas.

178. SOC (Security Operations Center)

Centro especializado en la monitorización y respuesta a incidentes de ciberseguridad en una organización.

179. Criptominería

Proceso de usar potencia computacional para validar transacciones y crear nuevas monedas en una blockchain.

180. Hash
